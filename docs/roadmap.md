# Booking-Ops Roadmap

## Milestone 0 — Foundation
- Repo structure
- Vision + roadmap docs
- GitHub issues + project board

## Milestone 1 — AI Extraction (Manual Input)
- Define booking JSON schema
- Build prompt for structured extraction
- Test extraction using pasted email text
- Save extracted booking to local JSON file

## Milestone 2 — Calendar Integration
- Connect Google Calendar API
- Check availability for extracted booking
- Create HOLD calendar event

## Milestone 3 — Database Integration
- Set up Supabase
- Store bookings in Postgres
- Query earnings totals by month

## Milestone 4 — Dashboard
- Display bookings list
- Monthly earnings chart
- Basic income forecast

## Milestone 5 — Gmail Automation
- Automatically detect booking emails
- Process without manual copy/paste
